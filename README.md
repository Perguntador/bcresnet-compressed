# BC-ResNet Compression Limits

Experimentos para determinar o limite inferior de compactação da arquitetura BC-ResNet para identificação de N comandos.

## Créditos e Atribuição

Este projeto é baseado na implementação oficial do **BC-ResNet** desenvolvida pela Qualcomm AI Research. O código original foi adaptado para fins acadêmicos e de experimentação.

**Repositório Original:** [Qualcomm-AI-research/bcresnet](https://github.com/Qualcomm-AI-research/bcresnet)
